# Namnamya
แอปพลิเคชันเตือนการรับประทานยา เป็นนวัตกรรมด้านสุขภาพที่ออกแบบมาเพื่อสนับสนุนการดูแลตนเองของผู้ป่วยอย่างเป็นระบบ โดยมีวัตถุประสงค์เพื่อเพิ่มความถูกต้องและสม่ำเสมอ ได้แก่ ระบบแจ้งเตือนการรับประทานยา สามารถกำหนดวัน เวลา และจำนวนยาตามแผนการรักษา พร้อมระบบแจ้งเตือนอัตโนมัติ เช่น การนัดพบแพทย์ การตรวจติดตามอาการ หรือการรับยา ระบบผู้ช่วยเสมือน
